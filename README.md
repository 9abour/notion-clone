# Notion Clone

## Features

> Sidebar

- Search for Page
- Settings to change the theme
- Trash to restore the pages
- List of pages

> Menu Bar

- Menu button (Modal)
  - Full width (toggle)
  - Hide sidebar (toggle)
  - Lock page
  - Delete page

> Page

- ### Cover
  - Change cover (Upload, Link)
  - Remove cover
- ### Page Info
  - Add icon
  - Add cover
  - Page title
- ### Editor

  - Input with placeholder of "Enter text or press / for commands"
  - ### After enter "/" open modal

    Basic blocks

    - Text
    - Page
    - Todo
    - Heading 1
    - Heading 2
    - Heading 3
    - Bulleted list
    - Numbered list
    - Toggle list
    - Quote
    - Divider

    Media

    - Image
    - Video
    - Audio
    - Code
